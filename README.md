# Enterprise-Market-Intelligence-Platform
Pipeline RAG end-to-end de inteligencia de mercado a gran escala. Implementa arquitectura de Big Data con PySpark para la ingesta y normalización (1FN) de 1.68M de reseñas en formato Parquet columnar. Indexa fragmentos semánticos en bases vectoriales de alta dimensionalidad y orquesta el contexto mediante LangChain y LLMs en Databricks.
